# Local data notes

Put **rasters and bulky downloads** under `data/rasters/` or in your ArcGIS Pro file geodatabase.
Those paths are gitignored — never commit them.

Recommended local (not in git):
- WRTC Housing Unit Risk/Exposure (MI / WI / MN)
- LANDFIRE EVT (clipped to AOI)
- PAD-US (if stored as GDB here)
- Optional: WHP, burn probability, plantations, infrastructure, mills
