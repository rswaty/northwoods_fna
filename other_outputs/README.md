
# "other_outputs" notes

These files are outputs used to create https://rswaty.github.io/northwoods/.  Code at https://github.com/rswaty/northwoods.git

| bps_aoi_attributes.csv | Biophysical Settings (historical ecosystems) for the aoi
| bps_aoi_disturbances.csv | Historical disturbance information including annual acres disturbed for the aoi
| mnfi_aoi_attributes.csv | Historical Mean Fire Return Interval per BpS including surface, mixed and replacement fire and annual probabilities
| evt_aoi_attributes.csv | Existing Vegetation Type information for the AOI
